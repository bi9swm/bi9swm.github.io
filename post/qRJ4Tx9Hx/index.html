[数通]<实验报告>配置Console用户界面的认证方式

2019-07-21

实验环境：eNSP模拟器下运行

  1. AAA认证
[Huawei]user-interface console 0
[Huawei-ui-console0]authentication-mode aaa

[Huawei]aaa
[Huawei-aaa]local-user admin password cipher admin
[Huawei-aaa]local-user admin service-type 
[Huawei-aaa]local-user admin service-type terminal
  1. Password认证
[Huawei]user-interface console 0
[Huawei-ui-console0]authentication-mode password 
[Huawei-ui-console0]set authentication password cipher admin
  1. none 无认证
[Huawei]user-interface console 0
[Huawei-ui-console0]authentication-mode none
